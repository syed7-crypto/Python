# 🧮 Python Calculator

A simple console-based calculator built in Python as a beginner project.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with division-by-zero handling)
- 🔼 Power (Exponentiation)
- 🧮 Modulus
- 📜 Calculation History
- 🖥️ Console Screen Clear
- 📋 Menu-driven interface

## Concepts Used

- Functions
- Loops (`while`)
- Conditional Statements (`if-elif-else`)
- Lists
- User Input
- String Formatting (f-strings)
- Python Modules (`os`)
- Git & GitHub Version Control

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/syed7-crypto/Python.git
```

2. Navigate to the project folder:

```bash
cd Python
```

3. Run the program:

```bash
python calculator.py
```
